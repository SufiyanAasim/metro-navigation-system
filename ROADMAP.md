# Roadmap

## v1.1.5 — Teal ✅ Shipped

- **Premium Custom Message Boxes.** Dark-themed `CustomMessageBox` replacing standard Windows dialogs.
- **Dedicated Printable Receipt Flow.** A clean, print-ready ticket screen with a custom Print button.
- **Custom Destination Select Buttons.** Graphical station pickers replacing dropdown combo boxes.
- **Restructured Options Menu.** Reordered wizard flow and simplified Stations page.

## Planned (unscheduled)

- Automated test suite covering `MetroNetwork`'s route calculation and `TripHistoryService`'s file I/O.
- Database-backed trip history (SQLite) as an optional upgrade path from the flat-file log.
- Additional metro lines/stations beyond the current 10-station network.

## Ongoing

- Community contributions via [CONTRIBUTING.md](CONTRIBUTING.md).
- Documentation upkeep as the codebase evolves.

---

Roadmap items are subject to change. Check [CHANGELOG.md](CHANGELOG.md) for what has shipped.
