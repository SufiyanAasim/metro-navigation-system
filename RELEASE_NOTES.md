# Release Notes: Metro Navigation System v1.0.0

Welcome to the initial release of the **Metro Navigation System**! This application is designed to help users plan routes and calculate travel costs within the Karachi Metro network.

## Release Metadata
- **Version**: `v1.0.0`
- **Release Date**: 2026-06-21
- **Platform**: Windows (x86/x64)
- **Target Framework**: .NET Framework 4.7.2

## Features Included
1. **Shortest Path Calculation**:
   - Leverages **Dijkstra's algorithm** to determine the optimal route with the minimum distance.
   - Shows the ordered sequence of stations to traverse.
2. **Interactive Metro Map**:
   - Integrates a visual map graphic representing the Karachi Metro network.
   - Interactive pins/markers highlighting the path.
3. **Trip Receipt & History**:
   - Generates customized trip receipts with the route details, total distance, and date/time.
   - Automatically stores trips in a local history log file located in the application directory under `Metro App Travel History/`.
4. **Station browser**:
   - Easily search and select stations directly from dropdown menus.

## System Requirements
- **Operating System**: Windows 7 SP1 / 8.1 / 10 / 11
- **Runtime**: [.NET Framework 4.7.2 or later](https://dotnet.microsoft.com/download/dotnet-framework/net472)

## How to Run the Application
1. Download the release archive `MetroNavigationSystem-v1.0.0.zip`.
2. Extract the ZIP file contents to a folder of your choice.
3. Double-click `Metro Navigation System.exe` to start the application.
